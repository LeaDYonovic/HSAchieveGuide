炉石成就攻略 使用说明

这是一个非官方的小工具，需要配合 Firestone 使用。
它主要读取 Firestone 在本地生成或导出的相关数据，用来更方便地查看炉石成就。

本工具与 Blizzard、Firestone、Zero to Heroes 均无官方隶属关系，仅供个人学习与交流使用。
本工具由社区个人整理与发布，开发过程中使用了 OpenAI 提供的辅助工具。
软件目前仍然有一些 bug，也漏了一部分成就；我最近没有继续细修，所以先放出一个能用版本。
请自行判断使用风险，并遵守 Hearthstone、Firestone 及相关服务的使用条款。


一、运行哪个文件

请直接运行：
HSAchieveGuide.exe

不要手动运行：
ExportMindVisionAchievements.v3.exe

这个文件是刷新数据时由主程序自动调用的辅助程序，正常使用时不需要单独打开。


二、基础使用办法

1. 先把压缩包完整解压到同一个文件夹里。
不要只拖出其中一个 exe 单独运行。

2. 双击 HSAchieveGuide.exe 启动。

3. 第一次打开时，程序会自动查找 Firestone 数据目录。
如果没找到，会提示你手动选择目录。
请选择包含这些文件的目录：
collection.json
achievements-completed.json
profile-achievements.json

4. 打开后，顶部按钮可以切换页面：
官方成就分类
我追踪的成就
我的卡牌收藏
使用说明

5. 需要更新数据时，点击右上角“刷新数据”。
如果炉石正在运行，程序会自动调用辅助导出程序刷新运行时数据。

6. 在“官方成就分类”里，可以先看汇总，再点“打开细分”查看具体成就。

7. 在“我的卡牌收藏”里，可以按职业、费用、类型等条件筛选卡牌。


三、常见情况

1. 如果打开后看不到数据，通常是 Firestone 数据目录没有定位正确。
重新启动程序，再按提示选择正确目录即可。

2. 如果刷新后数据没有变化，先确认 Hearthstone 是否正在运行。

3. 如果程序被安全软件拦截，请把整个解压后的文件夹加入信任，或者允许程序运行。
