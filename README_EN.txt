HSAchieveGuide - Quick Guide

This is an unofficial tool that is meant to be used together with Firestone.
It mainly reads data generated or exported locally by Firestone, so that Hearthstone achievements are easier to browse.

This tool is not officially affiliated with Blizzard, Firestone, or Zero to Heroes.
It is shared for personal learning and community use only.
This tool was organized and released by an individual community member, with OpenAI used as a development aid during the process.
Please note that the software still contains some bugs, and some achievements are still missing.
Use it at your own discretion, and please follow the terms of Hearthstone, Firestone, and related services.


1. Which file should I run?

Please run:
HSAchieveGuide.exe

Do not run manually:
ExportMindVisionAchievements.v3.exe

This file is a helper program automatically called by the main app when refreshing data.
In normal use, you do not need to open it yourself.


2. Basic usage

1. Extract the whole archive into one folder first.
Do not drag only one exe out and run it alone.

2. Double-click HSAchieveGuide.exe to start the app.

3. On first launch, the program will try to locate your Firestone data folder automatically.
If it cannot find it, it will ask you to choose the folder manually.
Please select the folder that contains files like:
collection.json
achievements-completed.json
profile-achievements.json

4. After launch, you can switch pages using the top buttons:
Official Achievement Categories
Tracked Achievements
My Card Collection
Instructions

5. When you want to update data, click "Refresh Data" in the top-right corner.
If Hearthstone is currently running, the program will automatically call the helper exporter to refresh runtime data.

6. In "Official Achievement Categories", you can first view the summary, then click "Open Details" to inspect specific achievements.

7. In "My Card Collection", you can filter cards by class, cost, type, and other conditions.


3. Common situations

1. If no data is shown after startup, the Firestone data folder is usually not set correctly.
Restart the program and choose the correct folder when prompted.

2. If the data does not change after refreshing, first make sure Hearthstone is actually running.

3. If your security software blocks the program, add the extracted folder to the allowlist or trust the program manually.
